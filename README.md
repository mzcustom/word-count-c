# word-count-c
Find the N most frequent words in a txt file

# Build
For Windows
> build.bat

for Linux
> ./build.sh

# Run
Executables are included. Clone this repo and inside the folder(directory),

for Windows, type
> q3 {N} or q3-simd {N}
 
and for Linux, type
> ./q3 {N} or ./q3-simd {N}

It prints a list showing from the 1st to the Nth most frequest words in the text.
